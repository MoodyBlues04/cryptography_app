Объяснение шифрования и дешифрования RSA

Главная
Реализация алгоритма

1. Генерация ключей

Ключи RSA генерируются с использованием двух различных больших простых чисел (p и q).

  • Модуль (n): n = p * q
  • Функция Эйлера (phi): phi = (p - 1) * (q - 1)
  • Публичный ключ (e): Число, взаимно простое с phi, 1 < e < phi
  • Приватный ключ (d): Обратное для e по модулю phi. То есть: d * e = 1 (mod phi)

2. Шифрование

Чтобы зашифровать сообщение:

  1. Преобразуйте сообщение в байты.
  2. Для каждого байта вычислите его хэш ciphertext = byte ^ e (mod n).
  3. Сохраните зашифрованный текст (ciphertext для каждого байта) в виде строки, разделенной пробелами.

3. Дешифрование

Чтобы расшифровать зашифрованный текст:

  1. Разделите зашифрованный текст на части.
  2. Для каждой части вычислите расшифрованный текст plaintext = chunk ^ d (mod n) (где chunk - зашифрованный блок текста).
  3. Преобразуйте расшифрованный текст (plaintext) обратно в байты и декодируйте его в строку.
