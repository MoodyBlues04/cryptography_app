RSA Encryption and Decryption Explanation

This page explains the RSA encryption and decryption process based on the provided Python code.

1. Key Generation

RSA keys are generated using two large prime numbers (p and q).

  • Modulus (n): n = p * q
  • Euler's Totient (phi): phi = (p - 1) * (q - 1)
  • Public Key (e): A number coprime with phi.
  • Private Key (d): The modular inverse of e modulo phi.

2. Encryption

To encrypt a message:

  1. Convert the message into bytes.
  2. For each byte, compute ciphertext = pow(byte, e, n).
  3. Store the ciphertext as a space-separated string.

3. Decryption

To decrypt the ciphertext:

  1. Split the ciphertext into chunks.
  2. For each chunk, compute plaintext = pow(chunk, d, n).
  3. Convert the plaintext back into bytes and decode it into a string.
