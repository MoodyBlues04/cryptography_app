Объяснение шифрования и дешифрования RSA

Реализация алгоритма

1. Генерация ключей

Ключи RSA генерируются с использованием двух больших простых чисел (p и q).

  • Модуль (n): n = p * q
  • Функция Эйлера (phi): phi = (p - 1) * (q - 1)
  • Публичный ключ (e): Число, взаимно простое с phi.
  • Приватный ключ (d): Модульное обратное число для e по модулю phi.

2. Шифрование

Чтобы зашифровать сообщение:

  1. Преобразуйте сообщение в байты.
  2. Для каждого байта вычислите ciphertext = pow(byte, e, n).
  3. Сохраните зашифрованный текст в виде строки, разделенной пробелами.

3. Дешифрование

Чтобы расшифровать зашифрованный текст:

  1. Разделите зашифрованный текст на части.
  2. Для каждой части вычислите plaintext = pow(chunk, d, n).
  3. Преобразуйте расшифрованный текст обратно в байты и декодируйте его в строку.
